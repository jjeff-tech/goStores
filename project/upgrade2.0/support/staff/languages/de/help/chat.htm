Chat
 Vorherige  


Chat-Bereich ist, wo ein Mitarbeiter mit dem Nutzer / Kunden und andere Mitarbeiter chatten können.
Sie können zur Liste hinzufügen und bearbeiten Konserven Nachrichten von Mitarbeitern in den Live Chat nutzen. Suche von Canned Nachrichten ist auch possible.The "Chat-Protokolle" verlinken werden alle Chat-Protokolle, dass die Mitarbeiter anzuzeigen. Sie können die Protokolle durch die Eingabe zwei beliebige Datum-Zeit-Bereichen zu suchen. Außerdem können Sie die Chat-Protokolle.

    Sie können den Chat-Fenster mit einem Klick auf den Start Chat Link des Statuts Bereich zu starten. Dann wird das Chat-Fenster öffnet sich, wo man Sie Chat-Operationen.
   Am unteren Rand des Chat-Fensters gibt es eine tabbed section.The ersten Registerkarte werden alle Online-Personal und Anwendern zur Verfügung zu plaudern. Wenn der Mitarbeiter will sich mit einer der aufgeführten Mitarbeiter / users Chat, auf den "Chat"-Taste, die in das letzte Feld der spezifischen Zeile klicken. Die Anfrage wird an den ausgewählten Benutzer gesendet werden oder Mitarbeiter.Der zweite Registerkarte listet alle ausstehenden Chats.
    Der dritte Reiter listet alle Chats, die durch die in Personal protokolliert übertragen werden.
    Die Chat History Register listet die gesamte Chat-Protokoll des Personals. Der Benutzer info Reiter listet die Benutzer, die gerade zu Besuch sind die Web-Seite des Unternehmens. Hier werden Informationen wie die IP-Adresse des Besuchers Maschine, besucht die Seite durch den Benutzer etc. Wenn der Mitarbeiter will die Benutzer einzuleiten, um ein Schwätzchen mit ihm zu haben, auf dem "Invite Chat"-Taste entspricht, dass bestimmte Zeile klicken. Dann automatisch, wird das Chat-Besucher erhalten eine Benachrichtigung für Chat.
    Nach dem Senden einer Chat-Anfrage, wenn der Benutzer / Mitarbeiter am anderen Ende den Antrag an, ein Pop-up-Fenster mit dem Namen der Mitarbeiter in den linken Bereich des Chat-Fensters angezeigt werden. Wählen Sie im Popup-Fenster und chatte mit, dass die Mitarbeiter / user, indem Sie Ihre Chat-Text in den Textbereich zur Verfügung gestellt. Die Schriftarten wie Schriftfarbe, Schriftart und Schriftgröße können auch auf der Chat-Text angewendet werden. Sie können die Taste "Enter"-Taste oder klicken Sie auf den Senden-Button nach Eingabe der Chat-Text, um ihn zu senden.
    Wenn ein Benutzer oder Mitarbeiter zu senden Anfragen an einen anderen Mitarbeiter für den Benutzer, ein Pop-up-Fenster mit einem "Übernehmen"-Taste wird im linken Bereich des Chat-Fensters zu kommen. Wenn wir also auf die Anfrage annehmen wollen, auf die "Übernehmen"-Taste des jeweiligen Popup-Fenster klicken.
    Option für die Übertragung von einem Chat zu einem anderen Mitarbeiter möglich ist. Wenn ein Mitarbeiter will einer der chattet er zu einem anderen Mitarbeiter Handhabung ist Transfer auf der "Transfer-Chat"-Symbol des Chat-Fensters klicken. Das Popup-Fenster mit einer Liste der Mitarbeiter und ein 'Transfer Chat-Button kommen. Klicken Sie auf die Schaltfläche für das Personal, die den Chat übertragen werden soll. Dann wird automatisch wird der Chat auf, dass insbesondere Mitarbeiter übertragen werden.
    Das Hauptmerkmal der Live-Chat ist Remote Desktop Sharing. Wenn der Mitarbeiter will der Benutzer, Computer-Desktop, um mit ihm gemeinsam genutzt werden, den Benutzer auffordern, seine Desktop-Share. Wenn die Benutzer Desktop ist bereit zu teilen, können Mitarbeiter auf dem "Desktop-Share"-Symbol klicken, um die Benutzer-Desktop-Ansicht.
