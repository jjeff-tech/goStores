Sie können die Details aller Anwender im Unternehmen / Firmen, die Sie zugewiesen sind. Sie können auch E-Mails an die Benutzer. Darüber hinaus können Sie E-Mails an ausgewählte Nutzer auch zu senden. Um mehrere E-Mail-Adressen wählen, halten Sie die Strg-Taste und wählen Sie die Adressen eins nach dem anderen.
