Tickets Abschnitt zeigt alle Tickets mit Ihnen besorgt. Sie können im oberen Teil, alle Tickets vergeben, dich zu sehen. Darunter können, um die Liste aller neuen Karten sehen. Klicken Sie auf die Ansicht-Symbol, um das Ticket zu sehen. Sie können die Tickets Eigenschaften wie die ID, Eigentümer, Abteilung, Datum der Erstellung etc. Sie sehen auch die persönlichen Notizen von Mitarbeitern zu jedem Ticket aufgenommen. Außerdem können Sie das Feedback von den Kunden eingegeben worden sind. Die persönlichen Notizen auf einem Ticket kann durch andere Mitarbeiter haben auch werden. Sie können antworten, zitieren und beantworten oder Kommentar gegen ein Ticket. Sie können auch erweiterte Suche für Ihre Karten. Advanced Search können Unternehmen tun, Abteilung, Status, Besitzer, User, Ticket-Nummer, Titel und / oder die Beschreibung.
