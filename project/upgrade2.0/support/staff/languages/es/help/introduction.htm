El sistema de apoyo iScripts es un sistema integrado para la gestión de la consultas de los clientes, las respuestas y las comunicaciones como resultado de dichas consultas y problemas relacionados con la ayuda de una serie de herramientas e instalaciones agrupadas en torno al sistema de gestión de entrada como el sistema de base de conocimientos, el sistema de recordatorio, de correo y sistema de mensajería, correo electrónico y sistema de tuberías para mencionar unos pocos.

Los usuarios potenciales del sistema son los usuarios (clientes), personal (técnicos), y los administradores.

Los usuarios / clientes se comunican con el sistema para conseguir sus problemas y preguntas resuelto. Por lo general, buscar en la correspondiente sección de preguntas frecuentes para ver si un problema similar (y solución) está en la lista ya. Si el usuario no es capaz de encontrar alguna coincidencia, s / que envía un ticket con los detalles necesarios. El personal (técnicos) responder a las entradas con una respuesta predefinida o una de nueva creación, de acuerdo con la situación. La principal funcionalidad del sistema es la comunicación de ida y vuelta entre el usuario y el personal. En el caso de un equipo es incapaz de resolver un problema de s / he puede escalar el problema al administrador.

Otra característica importante del sistema de apoyo es iScripts LiveChat. Además del sistema de gestión de entrada, los clientes pueden chatear con los operadores para cualquier duda. Instalación entre conversación con el operador también está disponible en este sistema. Si es necesario, los clientes pueden compartir su computadora de escritorio con el operador al que está conversando través de la función Escritorio remoto compartido.Esto le ayudará a resolver sus problemas con facilidad. Los clientes pueden seleccionar un determinado departamento de la empresa y es capaz de conversar con el personal de ese departamento en particular.
