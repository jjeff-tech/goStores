Las plantillas son respuestas predefinidas, que se almacenan para su reutilización en cualquier momento. Las plantillas se pueden definir y almacenar una vez, y cuando te encuentras con el mismo problema / situación, sólo se puede seleccionar, su respuesta se rellenará con la respuesta predefinida. En algunos casos en la configuración del software requiere plantillas para ser aprobado por el administrador, no podrá usar la plantilla que acaba de agregar. A continuación, puede tener que esperar a la aprobación del administrador. Una vez que el administrador apruebe la plantilla, o cualquier otra persona podrá utilizar la plantilla yor.
