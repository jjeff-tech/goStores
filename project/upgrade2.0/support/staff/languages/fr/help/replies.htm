Alors que vous répondez à un billet, vous pouvez sélectionner votre réponse à partir d'un modèle prédéfini et / ou ajouter votre réponse personnalisée. Vous pouvez ajouter des commentaires sur le billet sur ​​lequel vous travaillez, ce commentaire sera affiché à vous seul. Alors que l'affichage de votre réponse, si vous trouvez votre réponse mérite d'être ajouté comme une nouvelle entrée dans la base de connaissances, sélectionnez la catégorie correspondante et cochez la case "Ajouter à Ko 'case à cocher. Dans l'onglet d'action, vous pouvez définir le nouveau statut du billet et d'opter pour avertir l'utilisateur avec un e-mail. Vous pouvez également revendiquer la propriété du billet. En outre, vous pouvez aussi verrouiller le billet de sorte que maintenant que vous pouvez gérer la correspondance et d'autres opérations avec ce billet. Avec un billet verrouillé, autres membres du personnel ne peut afficher les détails, ils ne pouvez pas répondre / commenter à elle. Dans la section «Autres informations», si vous voulez une copie carbone de l'e-mail envoyé, vous pouvez ajouter l'adresse e-mail / adresses (séparées par une virgule) dans le champ CC. Le temps passé sur ce billet dans cette session particulière doit être entré ici à des fins d'évaluation comptable et de performance. Vous pouvez également ajouter une pièce jointe (s) pour votre réponse.
