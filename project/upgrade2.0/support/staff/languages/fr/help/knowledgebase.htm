Les questions les plus fréquemment posées et les réponses sont stockées dans la base de connaissances. Chaque département aura une structure de catégorie de ses propres, et les entrées de base de connaissances sera ajouté dans la catégorie la plus appropriée. Vous pouvez afficher et ajouter des entrées. KB entrées seront ajoutées que dans les catégories des feuilles. Pour ajouter une catégorie, vous devez contacter l'administrateur. Dans le cas des paramètres des systèmes de demander à l'administrateur d'approuver chaque entrée Ko, vous ne serez pas en mesure de voir l'entrée que vous avez posté. Une fois que l'administrateur approuve les entrées, vous pouvez voir / modifier les.
