Modèles
 précédente  Suivant

Les modèles sont des réponses prédéfinies, qui sont stockés pour une réutilisation à tout moment. Les modèles peuvent être définis une fois et stockés, et quand vous tombez sur le même problème / scénario, il vous suffit de sélectionner, votre réponse sera rempli avec les réponses prédéfinies. Dans certains cas où les paramètres logiciels nécessitent des modèles pour être approuvé par l'administrateur, vous ne serez pas en mesure d'utiliser le modèle que vous venez d'ajouter. Vous pouvez alors avoir à attendre l'approbation de l'administrateur. Une fois que l'administrateur approuve votre modèle, vous ou quelqu'un d'autre pourra utiliser le modèle de yor.
