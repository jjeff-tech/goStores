La section Billets affiche tous les billets concernés avec vous. Vous pouvez voir dans la partie supérieure, tous les billets qui vous sont assignées. En dessous, vous pourrez voir la liste de tous les nouveaux billets. Cliquez sur l'icône afin de visualiser le billet. Vous pouvez voir les propriétés du ticket, comme l'ID, le propriétaire, département, date de création etc Vous pouvez également voir les notes personnelles ajoutées par les membres du personnel sur chaque billet. Vous pouvez également voir les commentaires postés par l'utilisateur. Les notes personnelles sur un billet peuvent être consultés par le personnel d'autres aussi. Vous pouvez répondre, devis et réponse ou un commentaire sur un billet. Vous pouvez également effectuer Recherche avancée de vos billets. Recherche avancée peut être fait sur la société, Département, Etat, propriétaire, utilisateur, numéro de référence du ticket, le titre et / ou la description.
