Vous pouvez personnaliser vos écrans en fonction de vos préférences dans cette section. En outre, vous pouvez modifier votre profil de changer vos identifiants, et vous pouvez également visionner vos logs d'activité pour voir vos opérations dans un intervalle de dates spécifié. Dans la section Attribuer les champs, vous pouvez ajouter des colonnes supplémentaires dans votre page listant les billets. Par exemple, si vous voulez ajouter ie 'User Name'., Le nom de l'utilisateur qui a créé un ticket, vous pouvez déplacer l'option «Nom d'utilisateur» de la disposition de la liste sélectionnée et enregistrer vos paramètres.
Vous pouvez également modifier votre profil de changer vos identifiants et vos préférences. Les paramètres que vous pouvez choisir sont: 1. Dispositif d'affichage, 2. Auto taux de rafraîchissement, 3. Que ce soit pour notifier par courriel quand un ticket vous est attribué, 4. Que ce soit pour notifier par courrier électronique un message privé vous est envoyé, 5. Que ce soit pour notifier par courriel quand une entrée de nouvelles connaissances de base est ajouté au système, dans les départements et donc les entreprises qui vous ont accès à.
